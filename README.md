# Basic-HTML-Website-Roadmap.sh

### Basic HTML Website

Repositori ini berisi implementasi proyek **Basic HTML Website** dari [roadmap.sh](https://roadmap.sh/projects/basic-html-website). Fokus utama proyek ini adalah membangun fondasi situs web multi-halaman menggunakan HTML murni dengan penekanan pada struktur semantik dan kesiapan pengembangan lebih lanjut.

#### Fitur Utama

* **Struktur Multi-Halaman**: Terdiri dari empat halaman utama: *Homepage, Projects, Articles,* dan *Contact*.
* **Navigasi Global**: Navigasi konsisten yang menghubungkan seluruh halaman situs.
* **HTML5 Semantik**: Penggunaan elemen `<header>`, `<nav>`, `<main>`, `<section>`, dan `<footer>` untuk aksesibilitas tinggi.
* **Optimasi SEO**: Penggunaan Meta Tags pada setiap halaman untuk meningkatkan keterbacaan oleh mesin pencari.
* **Ready for Styling**: Penataan elemen menggunakan ID dan Class yang terstruktur untuk kemudahan integrasi CSS nantinya.

#### Daftar Halaman

* `index.html`: Beranda utama dengan perkenalan singkat.
* `projects.html`: Daftar proyek yang telah dikerjakan.
* `articles.html`: Ruang untuk publikasi artikel atau tulisan.
* `contact.html`: Informasi kontak untuk keperluan komunikasi.

#### Cara Menjalankan

1. Clone repositori ini:
```bash
git clone https://github.com/bagusadesaputra/Basic-HTML-Website.git

```


2. Buka direktori proyek:
```bash
cd Basic-HTML-Website

```


3. Jalankan file `index.html` pada browser favorit Anda.

#### Tujuan Pembelajaran

Proyek ini dikerjakan untuk memahami cara mengelola konten web secara logis, membangun hierarki informasi yang benar, dan menerapkan standar dasar struktur web sebelum memasuki tahap desain visual.
